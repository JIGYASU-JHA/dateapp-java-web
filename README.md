# DateApp Java Servlet Application

A simple Java Servlet based web application developed using Apache Tomcat.

This project demonstrates how Java Servlets process HTTP requests and generate dynamic responses.

## Features
- Displays current date and time
- Servlet-based request handling
- Dynamic web response generation
- Java web application structure using Tomcat

## Technologies Used
- Java
- Servlet API
- Apache Tomcat

## Project Structure

DateApp/
│
└── WEB-INF/
     ├── web.xml
     └── classes/
          └── DateSrv.java

## How to Run

1. Install Apache Tomcat
2. Copy the project folder into the `webapps` directory
3. Start the Tomcat server
4. Open the application in browser

Example:

http://localhost:8080/DateApp

## Learning Outcomes
- Understanding Java Servlets
- Handling HTTP requests and responses
- Web application deployment using Apache Tomcat
- Working with web.xml configuration
